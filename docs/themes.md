# OTRM Themes

We have generated a few default themes that allows you to customize and style your terminal easily.

## Default Themes

### OTRM (OTRM)

```js
{
    theme: Themes.OTRM,
}
```

![theme-otrm](https://github.com/DERCHUB/otrm/blob/master/docs/otrm-theme-otrm.png?raw=true)
